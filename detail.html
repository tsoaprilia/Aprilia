The Client Who we worked with

The Bindawaood application is a mobile application that can help during the Hajj / Umrah activities take place. This application also contains the BinDawood website that will make your hajj and umrah pilgrimage calm.

02

Problems The main obstacles we faced

People find it difficult to find a trusted travel agent for hajj and umrah activities. Due to the number of fraud cases, the public tends to be less confident in doing Hajj and Umrah activities.

03

Solutions How we solved the challenges

Create an application that offers hajj and Umrah travel activities, tickets, hotels, food and others. This application also provides a Hajj and Umrah pilgrimage guide which can start from booking tickets, room and other prayer, the nearest mosque and others.
